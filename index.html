About This Project

OrangeCrypt was developed as an experimental project to explore Windows file system security techniques. It demonstrates how folder encryption can be implemented at the application level.

This project serves as a learning resource for developers interested in security implementations and Windows application development. It is not intended as production-ready software.

The source code is available on GitHub for educational examination and research purposes. Contributions and feedback from the developer community are welcome.
