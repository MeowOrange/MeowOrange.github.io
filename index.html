OrangeCrypt

OrangeCrypt is an experimental/conceptual folder encryption tool for Windows. It explores concepts of data security through simple encryption techniques.

Important Note: This is purely an experimental project created for educational and conceptual purposes. It is not intended for actual use as a security solution. Do not rely on this tool for real data protection.
